# Boho Homepage Asset Ledger

Purpose: record the sources, license posture, and implementation decisions for the experimental Boho homepage pass before source components are changed.

## Decisions

- Icon set: use the already installed Tabler Iconify package for UI icons; do not add a second icon pack for this pass.
- Animation library: install GSAP and use only the public npm package for small SVG/CSS entrance and motion effects.
- Font: add Space Grotesk through Fontsource for display headings; keep Inter for body copy.
- Pattern strategy: build honeycomb, radar, node graph, and incident-board visuals as custom inline SVG/CSS in repo-owned components.
- Asset download policy: no downloaded art, no copied template code, no screenshots embedded into the homepage.

## Install Recommendations

Run after this ledger checkpoint:

```powershell
npm install gsap @fontsource-variable/space-grotesk
```

Do not install for this pass:

- `animejs`
- `@iconify-json/ph`
- `@fontsource-variable/archivo`
- Three.js, Framer Motion, React, Vue, Svelte, or UI kits

## Implementation Guardrails

- The homepage should use custom components and repo-owned SVG/CSS systems.
- Keep motion decorative, not required for comprehension.
- Respect `prefers-reduced-motion`.
- Do not add stock images, template assets, screenshots, fake awards, fake metrics, fake client logos, or copied commercial code.
